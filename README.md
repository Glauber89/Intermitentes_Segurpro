# ⛏️ MineGuard - Sistema de Controle de Colaboradores

Sistema web responsivo para gestão de colaboradores de segurança em operações de mineração, com sincronização em tempo real via Firebase.

## ✨ Funcionalidades

- 📊 **Dashboard** — Visão geral por status e por mina
- 👥 **Cadastro de Colaboradores** — Feristas, Mensalistas, Intermitentes
- 📍 **Controle de Alocação** — Onde cada um está e o que está fazendo
- 🔄 **Gestão de Coberturas** — Quem está cobrindo quem e onde
- 🔥 **Tempo Real** — Múltiplos usuários simultâneos com Firebase
- 📱 **Responsivo** — Mobile e Desktop

## 🏗️ Minas/Unidades

- Cauê
- Conceição
- Periquito
- Projetos Capital
- BD

## 🛡️ Funções

- Porteiro
- Vigilante Móvel
- Vigilante Fixo

## 🚀 Como Usar

1. Clone o repositório
2. Crie um projeto no [Firebase Console](https://console.firebase.google.com)
3. Ative o **Realtime Database** (modo de teste)
4. Registre um app Web e copie as credenciais
5. Cole as credenciais no arquivo `firebase-config.js`
6. Abra `index.html` no navegador
